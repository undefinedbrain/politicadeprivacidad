# Política de Privacidad

Última actualización: 22 de abril de 2026

Esta Política de Privacidad describe cómo se maneja la información en esta aplicación.

## Recopilación de información

Esta aplicación no recopila directamente información personal identificable de los usuarios. Sin embargo, puede utilizar servicios de terceros que sí recopilan información utilizada para identificarte.

## Servicios de terceros

La aplicación utiliza servicios de terceros que pueden recopilar información para identificarte.

Entre ellos se incluye:

* Google AdMob (publicidad)

Google AdMob puede recopilar datos como identificadores de dispositivos, dirección IP, información sobre el uso de la aplicación y otros datos con fines publicitarios.

Para más información, consulta la política de privacidad de Google:
https://policies.google.com/privacy

## Uso de la información

Los datos recopilados por servicios de terceros se utilizan para:

* Mostrar anuncios personalizados o no personalizados
* Mejorar la experiencia del usuario
* Analizar el uso de la aplicación

## Cookies e identificadores

Los servicios de terceros pueden utilizar cookies o identificadores similares para recopilar información y ofrecer anuncios.

## Conexión a Internet

Esta aplicación requiere conexión a Internet para mostrar anuncios y otras funcionalidades.

## Seguridad

Se toman medidas razonables para proteger la información, aunque ningún método de transmisión por Internet es 100% seguro.

## Privacidad de los niños

Esta aplicación no está dirigida a menores de 13 años. No se recopila información personal de forma consciente de niños menores de 13 años.

## Cambios en esta política de privacidad

Esta Política de Privacidad puede actualizarse ocasionalmente. Se recomienda revisar esta página periódicamente para cualquier cambio.
